# Image Directory Structure

This directory contains all local images for the website. All images must be local (no external sources).

## Directory Structure

```
public/images/
├── hero/
│   ├── hero-main.jpg          # Main hero background image
│   ├── admission-banner.jpg   # Admissions page banner
│   ├── library-banner.jpg     # Library page banner
│   └── contact-banner.jpg      # Contact page banner
├── about/
│   ├── school_building.jpg     # School building image
│   ├── students_in_class.jpg   # Students in class
│   └── morning_assembly.jpg   # Morning assembly
├── academics/
│   ├── science_lab.jpg        # Science laboratory
│   ├── commerce_class.jpg     # Commerce class
│   ├── arts_studio.jpg        # Arts studio
│   └── ict_lab.jpg            # ICT lab
├── gallery/
│   ├── science_fair.jpg       # Science fair event
│   ├── graduation_day.jpg     # Graduation ceremony
│   ├── sports_day.jpg         # Sports day
│   └── cultural_day.jpg       # Cultural festival
└── misc/
    └── placeholder_profile.jpg # Placeholder profile image
```

## Adding Images

1. Add your images to the appropriate folders above
2. Use descriptive, lowercase filenames with underscores or hyphens
3. Recommended formats: `.jpg`, `.png`, `.webp`
4. Optimize images before adding (compress for web)

## Current Status

**Note:** These directories have been created, but you need to add actual image files to them. Until images are added, you will see 404 errors in the browser.
